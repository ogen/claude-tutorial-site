    第五章：個人化設定¶

    Memory 記憶系統¶

    每次開新對話，Claude 預設是一張白紙——它不記得你上次說你是工程師、用 Python、偏好繁體中文。Memory 系統解決的就是這件事，讓 Claude 跨對話記住你的個人資訊，不用每次都重新自我介紹。

    記憶的運作方式¶

    • Claude 會從對話中自動萃取重要資訊儲存
    • 下次對話時自動載入相關記憶
    • 你可以主動要求更新或刪除記憶

    主動設定記憶¶

    不用等 Claude 自己發現，直接告訴它你想讓它記住的事：

    請記住：我是一位台灣的軟體工程師，
主要使用 Python 和 JavaScript，
偏好繁體中文，在台中工作。

    查看與管理記憶¶

    1. 點選右上角帳號圖示
    2. 選擇「Memory」
    3. 可查看、編輯、刪除已儲存的記憶

    記憶指令¶

    幾個常用的記憶操作，直接說給 Claude 聽就行：

    請記住 [資訊]          → 主動儲存
請忘記關於 [主題] 的記憶  → 刪除特定記憶
你記得我哪些事？         → 查看目前記憶

    Custom Instructions（自訂指令）¶

    如果 Memory 是讓 Claude 記住「你是誰」，那 Custom Instructions 就是讓 Claude 記住「你希望它怎麼回答你」。設定一次，之後所有對話都會自動套用。

    設定路徑¶

    設定 → Custom Instructions（或 User Preferences）

    建議設定範本¶

    以下是一個可以直接參考的範本，依你自己的情況調整：

    ## 我的背景
- 職業：軟體工程師 / 財務分析師
- 主要語言：繁體中文
- 技術偏好：Python、macOS 環境

## 回答偏好
- 使用繁體中文回答
- 程式碼範例使用 Python（除非我特別指定）
- 簡潔明確，避免冗長鋪陳
- 不確定的事情請明說，不要猜測

## 格式偏好
- 重要步驟使用編號清單
- 程式碼放在 code block 內
- 長文章加上標題分段

    語言與介面設定¶

    介面語言和回應語言是分開控制的，不要搞混：

    • 點選右上角帳號 → Settings → Language
    • 可設定介面語言（不影響 Claude 的回應語言）
    • 回應語言由對話內容決定，或在 Custom Instructions 指定

    主題設定¶

    視覺上的調整，找到你最舒適的閱讀環境：

    Settings → Appearance： - Light / Dark / System 三種模式 - 字型大小調整

    實作練習¶

    1. 設定 Custom Instructions：參考本章範本，撰寫屬於你自己的 Custom Instructions，至少包含：職業背景、常用程式語言（若有）、偏好的回答長度與語言。設定後開啟新對話，確認 Claude 是否按照指定偏好回應。

    2. 記憶管理：告訴 Claude「請記住：我目前正在學習 Claude 的完整使用方式，偏好有實際範例的說明風格」，然後進入帳號的 Memory 管理頁面確認記憶是否已建立，並觀察下一次對話是否有所不同。

    3. Project 與 Custom Instructions 搭配：建立一個「寫作助理」Project，在 Project 的 Custom Instructions 中設定「語氣偏正式，段落不超過 3 句話，避免使用 Emoji」，與全域 Custom Instructions 設定比較，觀察 Project 層級設定如何覆蓋全域設定。

    常見問題 Q&A¶

    Q：Memory 記憶在不同裝置（手機、電腦）之間同步嗎？ A：是的。Memory 儲存在你的 Anthropic 帳號中，登入同一帳號的所有裝置（網頁、iOS App、macOS App）都會共用同一份記憶。

    Q：Custom Instructions 和 Project 的 Custom Instructions 同時存在時，哪個優先？ A：Project 層級的 Custom Instructions 優先。在 Project 內對話時，Project 設定會覆蓋全域 Custom Instructions；在 Project 外的一般對話則只套用全域設定。

    Q：怎麼知道 Claude 記住了什麼？會記住敏感資訊嗎？ A：可以在帳號設定的 Memory 頁面查看所有已儲存的記憶並逐一管理。不建議讓 Claude 記住密碼、帳號資訊等敏感資料，必要時可主動要求「請忘記剛才提到的 xxx 資訊」。
