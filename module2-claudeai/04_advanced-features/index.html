    第四章：Claude.ai 進階功能¶

    Projects（專案）¶

    一般對話結束後，Claude 就忘了你說過什麼。Projects 解決的就是這個問題——它讓你建立一個有持久記憶與設定的專屬空間，適合需要長期持續推進的工作。

    建立 Project¶

    1. 左側側欄點選「New Project」
    2. 設定專案名稱與說明
    3. 上傳相關文件（作為長期知識庫）
    4. 設定 Custom Instructions（專案專屬指令）

    適合使用 Projects 的情境¶

    不是所有任務都需要 Project，但以下這幾種情境，有 Project 會省很多重複說明的時間：

    • 長期進行的工作項目（如：每週郵件摘要 Agent）
    • 需要 Claude 記住特定背景的任務
    • 團隊共享的知識庫（Team 方案）

    Project 的知識庫上傳¶

    支援上傳最多 20 份文件，Claude 在該 Project 內的所有對話都能參照這些文件。

    Artifacts（成品輸出）¶

    當你請 Claude 產生一份「可以獨立存在」的內容時——像是一份 HTML 頁面、一個 React 元件、一張流程圖——它會把成品顯示在右側的 Artifacts 面板，讓你直接預覽和互動，不用再複製貼上到其他地方。

    Artifacts 類型¶

    類型 說明 範例
    Markdown 格式化文件 報告、教學文件
    程式碼 可複製的程式碼區塊 Python、JavaScript
    HTML 可直接預覽的網頁 互動式圖表
    React 互動式元件 計算機、表單
    SVG 向量圖形 流程圖、示意圖

    觸發 Artifacts 的方式¶

    只要在 Prompt 中提到「產生可預覽的內容」，Claude 通常就會自動使用 Artifacts 輸出：

    請產生一個可下載的 HTML 報告...
幫我建立一個 React 元件...
請用 SVG 畫一張流程圖...

    Web Search（網路搜尋）¶

    Claude 的訓練資料有截止日期，但現實世界不停在變。Web Search 讓 Claude 能在回答前先去搜尋最新資訊，補上它「天生不知道」的部分。

    啟用方式¶

    輸入框右下角的搜尋圖示切換開關。

    適合使用時機¶

    • 查詢最新新聞、股價、法規
    • 確認近期事件
    • 搜尋最新技術文件

    Deep Research（深度研究）¶

    如果 Web Search 是「幫你查一下」，那 Deep Research 就是「幫你做一份完整的研究報告」。它會自動進行多輪搜尋與分析，適合需要全面調查的複雜主題。

    使用方式¶

    [啟用 Deep Research 模式]
請深度研究：台灣 AI 伺服器供應鏈在 2025 年的發展現況，
包含主要廠商、市場規模、與美中貿易衝突的影響。

    與一般搜尋的差異¶

    功能 Web Search Deep Research
    搜尋次數 1-3 次 10-20+ 次
    分析深度 基本摘要 多角度綜合分析
    完成時間 數秒 數分鐘
    輸出格式 對話式 結構化報告

    實作練習¶

    1. 建立你的第一個 Project：在 Claude.ai 建立一個名為「工作助理」的 Project，上傳一份你的工作文件（如 SOP、產品規格書），並設定 Custom Instructions 讓 Claude 以你的職稱與工作背景為前提回答。

    2. Artifacts 生成實驗：分別請 Claude 產生以下三種 Artifact，觀察右側面板的互動體驗：

    3. HTML 格式的個人履歷頁面
    4. SVG 格式的工作流程圖

    5. React 格式的簡單計算機元件

    6. Deep Research 比較：針對同一個主題（如「2025年台灣電動車市場」），分別用普通 Web Search 和 Deep Research 模式各提問一次，比較兩份輸出的深度、來源數量與報告結構。

    常見問題 Q&A¶

    Q：Project 和一般對話有什麼本質差異？ A：一般對話結束後記憶不保留；Project 中上傳的文件和 Custom Instructions 會永久作為背景知識，所有在該 Project 內的新對話都能自動引用，適合長期持續的工作任務。

    Q：Artifacts 產生的程式碼可以直接用嗎？ A：HTML 和 React Artifacts 可在 Claude.ai 右側面板直接預覽互動，但正式使用前仍建議在本地環境測試，特別是涉及資料輸入或外部 API 呼叫的功能。

    Q：Deep Research 產出的報告來源可靠嗎？ A：Deep Research 會列出引用來源，但仍建議對重要決策的資訊點擊來源連結自行驗證。Claude 擅長整合與分析，但資料正確性最終取決於網路來源品質。
