    第九章：Claude Code 環境準備與安裝¶

    什麼是 Claude Code？¶

    Claude Code 是 Anthropic 官方的命令列 AI 開發工具，讓你在終端機中與 Claude 協作寫程式。和網頁版最大的不同，在於它能直接接觸你的專案：

    • 讀取與修改你的程式碼
    • 執行終端機指令
    • 連結 MCP 工具
    • 載入 Skills 自動執行標準工作流

    如果你是開發者，這是你和 Claude 合作效率最高的方式。

    前置需求¶

    Claude Code 需要 Node.js 18 以上，先確認一下：

    # 確認 Node.js 版本（需要 18 以上）
node --version

# 如果未安裝或版本過舊，使用 nvm 安裝
curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.39.0/install.sh | bash
nvm install 22
nvm use 22

    Windows 用戶：macOS 的 nvm 腳本無法在 Windows 使用，請改用 nvm-windows：前往 https://github.com/coreybutler/nvm-windows/releases 下載 nvm-setup.exe 安裝後，在 PowerShell 執行 nvm install 22 與 nvm use 22。

    安裝 Claude Code¶

    npm install -g @anthropic-ai/claude-code

    啟動與登入¶

    進入你的專案資料夾，執行 claude 就可以開始：

    # 在你的專案資料夾中啟動
cd ~/your-project
claude

# 首次啟動會引導你登入
# 開啟瀏覽器完成 OAuth 授權即可

    確認安裝成功¶

    claude --version
# 應顯示版本號，例如：claude 2.x.x

    訂閱要求¶

    Claude Code 需要 Claude Pro 以上方案（$20/月）。 透過 claude.ai 登入即自動連結你的訂閱。

    基本互動模式¶

    Claude Code 有三種使用方式，依任務性質選擇最合適的一種：

    互動模式（預設）¶

    claude
# 進入對話介面，可持續對話

    非互動模式（單次執行）¶

    claude -p "請解釋這個資料夾的結構"
claude -p "請幫我寫一個 README.md"

    繼續上次對話¶

    claude --continue

    實作練習¶

    1. 完成安裝流程：按照本章步驟安裝 Node.js（使用 nvm）和 Claude Code，執行 claude --version 確認安裝成功，並記錄版本號。

    2. 第一次互動：進入你的任何一個專案資料夾（或建立空資料夾），執行 claude，輸入「你好，請列出這個資料夾中有哪些檔案，並說明整體結構」，觀察 Claude 如何閱讀並分析專案。

    3. 非互動模式練習：執行以下指令，比較互動模式與非互動模式的適用場景： 

      claude -p "請用一句話說明 Git 是什麼"
claude -p "請為一個 Python Flask 應用寫一份基本的 README.md"

    常見問題 Q&A¶

    Q：執行 npm install -g @anthropic-ai/claude-code 出現 permission denied，怎麼辦？ A：建議改用 nvm 安裝和管理 Node.js，nvm 安裝的 Node.js 不需要 sudo 權限。執行 nvm install 22 && nvm use 22 後再重試安裝指令即可。

    Q：Claude Code 需要 Pro 訂閱，Free 方案能不能用？ A：不行。Claude Code 需要 Claude Pro 方案以上（每月 $20 美元）。啟動 claude 後會自動偵測登入帳號的方案，若不符合要求會顯示升級提示。

    Q：claude --continue 可以繼續多久以前的對話？ A：Claude Code 會保留最近一次對話的記錄，只要沒有執行 claude --new 或 /clear，都可以用 --continue 接續。跨天或跨工作階段通常也能繼續，但若 context 過長會自動壓縮摘要。
