    第十八章：Claude API 基礎設定¶

    什麼時候需要用 API？¶

    Claude.ai 和 Claude Code 已經能解決大多數使用場景。但如果你有以下需求，就是時候直接呼叫 API 了：

    • 自行開發 Claude 應用程式
    • 批次處理大量文件
    • 整合進現有系統
    • 需要精確控制模型參數

    取得 API Key¶

    API Key 是呼叫 Claude API 的憑證，取得方式很簡單，但要注意它只顯示一次：

    1. 前往 https://console.anthropic.com
    2. 建立帳號（與 claude.ai 帳號不同）
    3. Settings → API Keys → Create Key
    4. 複製並妥善保存（只顯示一次！）

    費用說明¶

    Claude API 按 Token 計費，和 Claude.ai 訂閱是分開的兩套費用：

    模型 Input（百萬 tokens） Output（百萬 tokens）
    Haiku 較低 較低
    Sonnet 中等 中等
    Opus 較高 較高

    建議先在 console.anthropic.com 設定每月消費上限，避免測試期間產生意外費用。

    安裝 SDK¶

    選擇你熟悉的語言安裝對應的 SDK：

    Python¶

    pip install anthropic

    Node.js¶

    npm install @anthropic-ai/sdk

    設定 API Key¶

    API Key 不要寫死在程式碼裡，有兩種安全的做法：

    方法一：環境變數（推薦）¶

    # macOS — 加入 ~/.zshrc 或 ~/.bashrc
export ANTHROPIC_API_KEY="sk-ant-你的key"

# 重新載入
source ~/.zshrc

    Windows 用戶：有兩種方式。① 圖形介面：搜尋「環境變數」→「編輯系統環境變數」→「使用者變數」→「新增」，重開終端機生效。② PowerShell 永久設定：在 $PROFILE 加入 $env:ANTHROPIC_API_KEY = "sk-ant-你的key"，再執行 . $PROFILE 重新載入。詳細步驟見 Windows 差異說明。

    方法二：.env 檔案¶

    # .env
ANTHROPIC_API_KEY=sk-ant-你的key

    # Python 載入 .env
from dotenv import load_dotenv
load_dotenv()

    ⚠️ 安全注意事項¶

    API Key 一旦洩漏，任何人都可以用你的帳號呼叫 API 產生費用：

    # .gitignore 中加入
.env
*.key

# 永遠不要把 API Key commit 進 git

    實作練習¶

    1. 取得 API Key 並設定環境變數：前往 console.anthropic.com 建立帳號、產生 API Key，並依照本章方法一將 Key 加入 ~/.zshrc（或 ~/.bashrc）。重新載入後執行 echo $ANTHROPIC_API_KEY 確認設定成功。

    2. 設定消費上限：在 Anthropic Console 的 Billing 頁面設定每月消費上限（建議初學者設 $5-10 美元），避免測試時產生意外費用。截圖保存設定記錄。

    3. SDK 安裝驗證：執行以下指令安裝 Python SDK，並用一行 Python 確認 API Key 有效： 

      pip install anthropic
python -c "import anthropic; c = anthropic.Anthropic(); print('SDK 安裝成功，API Key 有效')"

    常見問題 Q&A¶

    Q：console.anthropic.com 帳號和 claude.ai 帳號是不同帳號嗎？ A：是的，兩者完全分開。console.anthropic.com 是開發者帳號，使用 API 按 token 計費；claude.ai 是消費者帳號，按月訂閱。費用不互通，API 呼叫不包含在 Claude Pro 訂閱中。

    Q：API Key 不小心 push 到 GitHub 怎麼辦？ A：立即在 console.anthropic.com 的 API Keys 頁面撤銷（Revoke）該 Key，再產生新的 Key。被撤銷的 Key 立即失效，但為安全起見也建議在 GitHub 的 Security 頁面掃描歷史 commit 確認沒有其他洩漏。

    Q：如何預估 API 費用？ A：Anthropic Console 提供 token 使用量報表。粗略估算：1000 個中文字約 1500-2000 tokens，一次完整的問答大約 1000-3000 tokens，Sonnet 的費用約 $0.003-0.009 美元。建議先設消費上限再開始測試。
