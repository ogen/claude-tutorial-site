    第六章：安裝 Claude Desktop¶

    什麼是 Claude Desktop？¶

    Claude Desktop 是 Anthropic 官方的桌面應用程式。表面上看起來和 Claude.ai 網頁版很像，但它有一個關鍵差異：支援 MCP（Model Context Protocol）工具整合，讓 Claude 能直接操作你電腦上的檔案、應用程式與外部服務。如果你希望 Claude 不只是「幫你想」，而是能「幫你做」，Claude Desktop 就是起點。

    macOS 安裝步驟¶

    1. 下載安裝檔¶

    前往 https://claude.ai/download，點選「Download for macOS」。

    2. 安裝¶

    # 方法一：直接開啟 .dmg 檔案拖曳安裝
# 方法二：使用 Homebrew
brew install --cask claude

    3. 登入¶

    開啟 Claude Desktop → 使用 Anthropic 帳號登入（與 claude.ai 同帳號）。

    4. 確認方案¶

    Claude Desktop 的 MCP 功能需要 Pro 方案以上。 可至 Settings → Account 確認目前方案。

    Windows 安裝步驟¶

    1. 下載¶

    前往 https://claude.ai/download，點選「Download for Windows」。

    2. 執行安裝程式¶

    執行下載的 .exe 安裝檔，依照指示完成安裝。

    3. 登入¶

    開啟應用程式 → 使用 Anthropic 帳號登入。

    確認安裝成功¶

    安裝完成後，進入 Settings 找到 Developer 頁面，是最快的驗證方式：

    • macOS：Claude 選單 → Settings → Developer
    • Windows：左下角齒輪 → Settings → Developer

    看到「MCP Servers」設定頁面即代表安裝成功，可以開始連結外部工具。

    Claude Desktop vs Claude.ai 差異¶

    大多數功能兩者都有，差別集中在 MCP 整合這塊：

    功能 Claude.ai Claude Desktop
    基本對話 ✅ ✅
    Web Search ✅ ✅
    Projects ✅ ✅
    MCP 工具整合 ✗ ✅
    本地檔案存取 ✗ ✅（透過 MCP）
    離線使用 ✗ ✗（仍需網路）

    實作練習¶

    1. 安裝 Claude Desktop：依照本章步驟完成安裝，進入 Settings → Developer，確認看到「MCP Servers」設定頁面。截圖存檔作為安裝成功的記錄。

    2. 確認方案與功能：登入後前往 Settings → Account，記錄目前的方案名稱，並確認是否符合使用 MCP 的最低需求（Pro 以上）。

    3. 體驗 Desktop 與網頁版差異：用相同的 Prompt 在 Claude.ai 網頁版和 Claude Desktop 各提問一次，觀察介面與功能上的差異，特別注意 Desktop 版的系統選單（Claude 選單 → Settings）比網頁版多了哪些選項。

    常見問題 Q&A¶

    Q：安裝 Claude Desktop 需要額外付費嗎？ A：下載和安裝本身免費，但要使用 MCP 工具整合功能需要 Pro 方案以上（$20/月）。Free 方案可以使用 Claude Desktop，但無法設定 MCP Server。

    Q：Claude Desktop 和我在瀏覽器用的 Claude.ai 是同一個帳號嗎？ A：是的，使用相同的 Anthropic 帳號登入，對話記錄、Projects 和 Memory 都會同步。

    Q：重新啟動電腦後 Claude Desktop 需要重新登入嗎？ A：通常不需要，登入狀態會保持。只有清除應用程式資料或主動登出時才需要重新登入。如果出現登入狀態異常，可嘗試在 Claude 選單 → Settings → Account 重新授權。
